# Changelog

이 파일은 git 히스토리와 별개로, 사용자에게 보여줄 릴리즈 노트를 정리하는 용도로 사용합니다.
과거 항목은 git 커밋 히스토리를 바탕으로 버전 단위를 추정해 백필했습니다.

## 2.1.0 - 2026-04-03

- 설정 페이지 하단에 현재 앱 버전을 표시하도록 추가했습니다.
- 앱 버전을 `version.js`에서 한 곳에서 관리하도록 정리했습니다.
- 서비스 워커 캐시 키가 앱 버전과 함께 갱신되도록 맞췄습니다.
- 릴리즈 노트 누적을 위한 changelog 파일을 추가했습니다.

## 2.0.0 - 2026-04-03

- Android WebView 환경에서 가이드 오버레이가 보이지 않던 문제를 수정했습니다.
- 네이티브 Android 설정 연동을 보강하고 설정 가이드 흐름을 더 안정적으로 다듬었습니다.
- 좁은 화면에서도 테스트 모드 토스트가 자연스럽게 보이도록 반응형 처리를 개선했습니다.

## 1.5.0 - 2026-04-02

- 진동과 테스트 모드를 추가하고, 테스트 모드가 앱 재시작 후에도 유지되도록 저장 방식을 보강했습니다.
- `TimestampTrigger`를 활용해 앱이 닫혀 있어도 OS 레벨 예약 알림이 가능한 환경을 지원했습니다.
- Android 네이티브 브리지(`NativeAlarm`) 연동을 추가해 래퍼 앱 대응 기반을 마련했습니다.
- 백그라운드 테스트 알람 충돌을 줄이고, 포그라운드/백그라운드 스케줄 동기화를 개선했습니다.
- 설정 가이드 위치를 다듬고 테스트 모드 안내 단계를 추가했습니다.

## 1.1.0 - 2026-04-01

- 설정 페이지, NTP 기반 타이머, 상태 알림을 포함한 큰 폭의 UI/구조 개편을 진행했습니다.
- 위치 정보를 이용한 일출/일몰 기반 낮/밤 시계 테마를 추가했습니다.
- 낮 테마가 적용되지 않던 `clockAmpm` 참조 문제를 수정했습니다.
- 처음 쓰는 사용자를 위한 가이드 레이어를 추가했습니다.

## 1.0.0 - 2026-03-31

- PWA 기반 정시 알람 앱의 첫 버전을 추가했습니다.
- 초기 저장소 정리와 기본 프로젝트 구성을 마무리했습니다.
