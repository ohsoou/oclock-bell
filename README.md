# 🔔 정시 알람 (O'Clock Bell)

매 정시마다 한국어 아기 목소리로 시각을 알려주는 Android PWA 웹앱.

> "열한시!" "두시!" — 귀엽고 정확하게.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🕐 정시 알람 | 설정한 시간 범위 안에서 매 정시마다 자동 발화 |
| 🗣️ 한국어 TTS | Web Speech API로 "한시~열두시" 발화, 음높이·속도·음량 조절 가능 |
| 🌐 NTP 시간 동기화 | 앱 시작 시 인터넷 시간 서버로 시계 오프셋 보정 |
| 🌙 오전/오후 테마 | 오전(밝음) / 오후(어두움) 자동 전환 |
| 🔔 상태 알림 | 알림창에 알람 켜짐/꺼짐 상태 상시 표시 (선택) |
| 📲 PWA | 홈 화면에 설치, 오프라인 동작 |
| 🔒 Wake Lock | 앱이 열린 동안 화면 꺼짐 방지 |

## 타이머 아키텍처

단순 `setInterval` 폴링 대신 세 가지 레이어를 조합해 정확도와 안정성을 확보합니다.

```
┌─────────────────────────────────────────────────────┐
│  timer.worker.js (별도 스레드)                        │
│                                                     │
│  1. NTP 동기화  →  시스템 시계 오프셋 계산             │
│  2. performance.now() 모노토닉 클럭으로 경과 시간 추적  │
│  3. setTimeout으로 다음 정시까지 정밀 예약              │
│     (폴링 없음 — 정시 순간에만 실행)                   │
└──────────────────┬──────────────────────────────────┘
                   │ ALARM 메시지
                   ▼
┌─────────────────────────────────────────────────────┐
│  app.js (메인 스레드)                                 │
│  TTS 발화 + Toast + 벨 애니메이션                     │
└──────────────────┬──────────────────────────────────┘
                   │ (백그라운드 전환 시)
                   ▼
┌─────────────────────────────────────────────────────┐
│  sw.js (Service Worker)                             │
│  showNotification() 으로 알림창 알람                  │
└─────────────────────────────────────────────────────┘
```

### 탭 스로틀링 방지

| 방법 | 역할 |
|------|------|
| Web Worker | 메인 스레드 분리 → OS 탭 스로틀 영향 감소 |
| Silent AudioContext | 무음 버퍼 재생으로 Chrome 탭 일시중단 방지 |
| Wake Lock API | 앱이 열린 동안 화면 슬립 방지 |
| Service Worker | 탭 백그라운드 시 알림 팝업 대체 |

> **한계**: 탭이 완전히 종료되면 웹 기술로는 알람 보장 불가.
> 완전한 백그라운드 지원은 Push 서버 또는 네이티브 앱이 필요합니다.

## 로컬 실행

Service Worker는 `file://` 프로토콜에서 동작하지 않으므로 로컬 서버가 필요합니다.

```bash
# Node.js
npx serve "oclock bell"

# Python
python3 -m http.server 8080 --directory "oclock bell"
```

브라우저에서 `http://localhost:3000` (또는 `:8080`) 접속

---

## 배포 (Netlify Drop)

1. [app.netlify.com/drop](https://app.netlify.com/drop) 접속
2. `oclock bell` 폴더 전체를 드래그 앤 드롭
3. 즉시 `https://xxxx.netlify.app` HTTPS URL 발급

HTTPS가 필수인 이유: Service Worker, Wake Lock, Web Notifications API 모두 보안 컨텍스트에서만 동작합니다.

## Android 설치 (PWA)

1. 안드로이드 크롬에서 배포 URL 접속
2. 주소창 우측 메뉴(⋮) → **"홈 화면에 추가"**
3. 홈 화면 아이콘으로 앱처럼 사용

## 설정 항목

### 알람 범위
- **시작 시간** / **종료 시간**: 알람이 울릴 시간대 (0~23시)

### 알림
- **상태 알림 표시**: 알림창에 알람 켜짐/꺼짐 상태를 항상 표시

### TTS
| 항목 | 범위 | 기본값 |
|------|------|--------|
| 목소리 | 시스템 한국어 음성 | 자동 |
| 음높이 (Pitch) | 0.5 ~ 2.0 | 1.5 |
| 말하기 속도 (Rate) | 0.5 ~ 1.5 | 0.8 |
| 음량 (Volume) | 0.0 ~ 1.0 | 1.0 |
