# 정시 알람 (O'Clock Bell)

설정한 시간 범위 안에서 정시마다 알림을 보내고, 앱이 열려 있을 때는 한국어 TTS로 시간을 읽어주는 PWA 웹앱입니다.

## 구현 상태

| 항목 | 현재 상태 |
|------|------|
| 실시간 시계 | 시/분/초 1초 단위 갱신 |
| 자동 테마 | 위치 기반 일출/일몰 계산, 위치 미확정 시 시간대 폴백 |
| 알람 범위 | 시작/종료 시각 설정, 자정 넘김 범위 지원 |
| 테스트 모드 | 1시간 간격 대신 1분 간격 |
| TTS | 음성 선택, 속도/피치/볼륨 조절, 테스트 재생 |
| 알림 권한 상태 | 설정 화면에 허용됨/차단됨/요청 전 표시 |
| 백그라운드 알림 | Service Worker + Notification 기반, 환경에 따라 부분 지원 |
| 실패 대응 | 알림 표시 실패 시 짧은 재시도 |

## PRD 대비 메모

- 앱이 열려 있는 동안에는 TTS 발화와 진동이 동작합니다.
- 앱이 완전히 종료된 상태에서는 웹 플랫폼 제약 때문에 TTS 재생을 보장할 수 없고, 가능한 경우 시스템 알림만 유지합니다.
- 무음/진동 모드의 실제 기기 상태를 웹 표준만으로 판별할 수 없어, 일반/진동 모드 분기 로직은 제공하지 않습니다.
- Push 서버 없이 동작하므로, 완전한 종료 상태 보장이 필요한 경우에는 Web Push 백엔드나 네이티브 래퍼가 추가로 필요합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 정시 알람 | 설정한 시간 범위 안에서 매 정시마다 알람 시도 |
| 한국어 TTS | Web Speech API로 시각 발화, 음높이·속도·음량 조절 가능 |
| NTP 시간 동기화 | 앱 시작 시 외부 시간 서버로 시계 오프셋 보정 시도 |
| 일출/일몰 테마 | 위치 기반으로 라이트/다크 자동 전환 |
| 상태 알림 | 알림창에 알람 켜짐/꺼짐 상태 상시 표시 가능 |
| PWA | 홈 화면 설치 및 오프라인 캐시 지원 |
| Wake Lock | 앱이 열린 동안 화면 꺼짐 방지 시도 |

## 타이머 아키텍처

단순 `setInterval` 폴링 대신 여러 레이어를 조합해 정확도와 안정성을 높입니다.

```text
timer.worker.js
  1. NTP 동기화로 시스템 시계 오프셋 계산
  2. 보정 시간 기준 1초 틱 생성
  3. setTimeout으로 다음 정시/정분 경계 정밀 예약

app.js
  1. 시계 UI 렌더링
  2. TTS 발화
  3. 설정 저장, 상태 표시, Service Worker 스케줄 전달

sw.js
  1. 앱 셸 캐시
  2. 예약 알림 표시
  3. 알림 표시 실패 시 짧은 재시도
```

## 백그라운드 동작

| 방법 | 역할 |
|------|------|
| Web Worker | 메인 스레드 분리, 1초 시계 갱신과 정시 경계 계산 |
| Silent AudioContext | 일부 브라우저에서 백그라운드 중단 완화 시도 |
| Wake Lock API | 앱이 열린 동안 화면 슬립 방지 |
| Service Worker | 백그라운드 시스템 알림 |
| TimestampTrigger | 지원 브라우저에서 예약 알림 강화 |

한계:
- 브라우저와 OS 정책에 따라 백그라운드 예약의 신뢰도는 달라집니다.
- 완전한 종료 상태 보장은 Push 서버 또는 네이티브 앱이 필요합니다.
- 앱 종료 후 음성 재생은 지원되지 않습니다.

## 로컬 실행

Service Worker는 `file://`에서 동작하지 않으므로 로컬 서버가 필요합니다.

```bash
npx serve "oclock bell"
```

또는

```bash
python3 -m http.server 8080 --directory "oclock bell"
```

## 설정 항목

### 알림
- 권한 상태: 허용됨 / 차단됨 / 요청 전
- 백그라운드 지원 상태: 지원 환경 여부 안내
- 상태 알림 표시: 알림창에 알람 켜짐/꺼짐 상태 표시

### TTS

| 항목 | 범위 | 기본값 |
|------|------|--------|
| 목소리 | 시스템 음성 목록 | 자동 |
| 음높이 | 0.5 ~ 2.0 | 1.5 |
| 말하기 속도 | 0.5 ~ 1.5 | 0.8 |
| 음량 | 0.0 ~ 1.0 | 1.0 |

### 테스트
- 테스트 모드: 정시 대신 1분 간격
- 목소리 테스트: 현재 설정으로 즉시 발화

## 제약 사항

- 웹앱은 기기의 무음/진동 모드 여부를 직접 감지할 수 없습니다.
- Push API 서버 연동은 아직 포함되어 있지 않습니다.
- 앱 종료 후 TTS는 불가능하며 시스템 알림 중심으로 동작합니다.
